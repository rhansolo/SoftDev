A few things:
  • The class grid-container defines a container in which "cells," or other divs with specific properties, should be placed.
  • The class grid-x grid-padding-x essentially acts as a new line within the grid-container, allowing placement of multiple cells on the same line.
  • As you've probably noticed, this div isn't attached to a grid-container. This is a div known as a callout, which is a colored element. There are multiple types:
    • primary callout
    • callout secondary
    • callout success
    • callout alert
    • callout warning
    Callouts can be attached to any cell or grid.
